# Using Tasks

## Create a task
- Open the **Tasks** page.
- Click **New Task**.
- Add a title (required) and optional details.
- Set a priority and optional due date.
- Click **Create**.

## Edit or delete a task
- Find the task in your list.
- Click the **Edit** (pencil) button to update title, details, priority, due date, or privacy mode.
- Click **Save** to keep changes.
- Click the **Delete** (trash) button to remove a task.

## Mark tasks as complete
- Click the checkbox on a task card.
- Completed tasks are visually marked so you can focus on what is left.

## Due dates and priorities
- Use **high priority** for urgent work.
- Add a **due date** when timing matters.
- Leave due date empty for flexible tasks.

## Examples
```text
Write a blog post (due Friday)
Read 10 pages of a book (daily)
```
